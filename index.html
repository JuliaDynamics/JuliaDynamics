JuliaDynamics

JuliaDynamics is GitHub organization that develops Julia packages dealing with nonlinear dynamics & chaos. These packages are tools used by scientists working in chaos and nonlinear dynamics.

Packages


An easy-to-use, modular, performant and extendable Julia package for dynamical billiards in two dimensions.


A Julia software library for the exploration of chaos and nonlinear dynamics.


Prediction of timeseries using methods of nonlinear dynamics and timeseries analysis.

Features

Simple

All packages are intuitive, simple to use and simple to understand. We also take extra care to write simple and consice source code.

Documentation

Every package is accompanied with its own dedicated documetnation which is automatically generated, always updated and full of details and examples. Click the logo of each package to access the documentation.

Open Source

All packages are open source and hosted on GitHub! Every package is also licenced under MIT, which is one of the most permissive licences. Want to contribute? Check out the GitHub repos of the individual packages!

Getting Started

All packages of JuliaDynamics are registered! To install them simply press ] to enter the package manager mode and then do: add PackageName. To get started with Julia, check out Julia's setup instructions. For plotting we recommend PyPlot or Makie.
